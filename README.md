# TAPIT
Typescript API Template.

## Requirements

- NodeJS ^12.16.3

## Install 

After cloning the repository, go to you folder 

```bash
npm install && npm install --only=dev
```

## Development

Run dev script 
```bash
npm run dev
```


## Compile for production

Compile application
```bash
npm run build
```

Start server
```bash
npm run start
```
